Joomla! 3.7 – Ausblick auf die neuen Funktionen

vielleicht schon im Dezember 2016

Erstellt von Astrid Günther

Danke

Das Menü-System

Legt man einen neuen Menüpunkt für einen einzelnen Beitrag an, hat aber vergessen, den Artikel zuvor zu erstellen, wird künftig der neue Button Erstellen (Create) dabei helfen, direkt aus dem Arbeitsfenster heraus, den fehlenden Artikel anzulegen. Dasselbe gilt auch in Bezug auf die anderen Menü-Typen, wie zum Beispiel Kategorie oder Kontakt usw.

Verbesserte Optionen-Anzeige

Joomla! bietet an vielen Stellen etliche Möglichkeiten um irgendwelche Einstellungen vorzunehmen. Legt man zum Beispiel einen neuen Artikel an, wird unter Optionen in den Drop-down-Menüs standardmäßig Globale Einstellung angezeigt. Von Vorteil ist es hier natürlich, wenn man sich an bereits vorgenommene Einstellungen erinnern würde. Statt nun in einem anderen Artikel gleicher Kategorie nachsehen zu müssen, soll in Joomla! 3.7 der gesetzte Status angezeigt werden.

Inhaltssprache wird mit installiert

Wer eine andere Sprache in Joomla! installiert, erhält diese für das Front- und das Backend, doch die Inhaltssprache muss manuell angelegt werden. Dies soll künftig automatisiert werden.

Custom Fields

Mit 3.7 sollen die Custom Fields Einzug in das CMS halten. Es sollen Feldgruppen und Felder (Bilder, Datum, Textfelder, usw.) angelegt werden können. Eine Feldgruppe kann einer Kategorie zugeordnet und allen, ein paar oder einem Artikel(n) hinzugefügt werden.

WYSIWYG-Editor individualisierbar

Wer den JCE-Editor nutzt, kennt die Möglichkeiten, die Icons und damit die verbundenen Funktionen des Texteditors individuell zusammenzustellen und dann einer oder mehreren Benutzergruppen zuzuweisen. Die Chancen, dass dieses Feature den Sprung in 3.7 schafft, stehen fifty-fifty.

Router

Vereinfachung für Programmierer beim Router in der Entwicklung. Für Joomla!-Nutzer gilt, es werden standardmäßig die vorhandenen URLs genutzt, aber man kann auf die Funktion des Routers pro Komponente umschalten und testen.
